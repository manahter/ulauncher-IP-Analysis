## Requirements

* [Ulauncher](https://github.com/Ulauncher/Ulauncher) 5.0+

## Install

Open ulauncher preferences window -> extensions -> add extension and paste the following url:

```
https://github.com/nesivmi/ulauncher-iplik
```

## Usage

* > ip
* > ip 8.8.8.8
* > ip google.com
* > ip mynet.com
