## Install

Open ulauncher preferences window -> extensions -> add extension and paste the following url:

```
https://github.com/manahter/ulauncher-IP-Analysis
```
```
```
## Usage

* > ip
* > ip 8.8.8.8
* > ip google.com
* > ip mynet.com
